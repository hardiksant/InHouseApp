# RudrakshWala Office

Complete business management platform for Rudraksha business operations.

## Features

- **ExpensePilot Module**: Track expenses, scan invoices, and manage company spending
- **Sales Bills**: Generate invoices, manage sales, and track payments
- **Product Library**: Manage inventory for Rudraksha beads and general products
- **CRM System**: Lead management, customer tracking, and order book
- **Astro Recommendation**: Generate personalized Rudraksha recommendations
- **Creatives Library**: Marketing materials and brand resources
- **Order Management**: Complete order workflow from creation to dispatch
- **User Management**: Role-based access control (Admin, Moderator, Sales)
- **System Reports**: Track bugs, improvements, and feature requests
- **Platform Reports**: Business analytics and insights

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Database & Authentication)
- PWA enabled for mobile installation

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables in `.env`

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## User Roles

- **Admin**: Full access to all modules including user management and system reports
- **Moderator**: Access to CRM, order management, and dispatch workflows
- **Sales**: Access to sales-related modules and customer management
